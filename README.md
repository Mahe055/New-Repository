AI MCQ Generator

This is a small Python project I made while learning about APIs and AI.
The program asks for a topic and then generates multiple choice questions using the OpenAI API.
I built this to practice:
- Working with APIs
- Python input/output
- Creating a simple CLI program

How to run it:

1. Install the required library
pip install openai
2. Add your OpenAI API key in the code
3. Run the file
python mcq_generator.py
Then enter any topic and the program will generate MCQ questions.
